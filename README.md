# Shopping Cart Offers

A React + Tailwind shopping cart that applies special offers dynamically.

## Offers Implemented
- Buy 1 get 1 free on Cheese
- Buy Soup → Bread at half price
- Butter at one-third off

## Tech Stack
- React (Vite)
- Tailwind CSS
- JavaScript

## Run Locally
npm install
npm run dev
